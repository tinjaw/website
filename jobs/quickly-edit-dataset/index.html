      Quickly Edit Dataset

      Quickly correct a value or a column.

      You want ways of quickly editing a dataset, or to collaborate on one, without having to setup a complete development environment.

      Context

      It isn't needed in every project. When it is, you may not have time to research the best tools.

      Although convenient, Excel may cause multiple issues when opening and saving tabular data. You're better doing it in tools more appropriate for data pipelines.

      # Examples

      Spreadsheets: Edit a cell in the sheet

      # Solutions

      Data Curator
      Edit CSV and XLS files without unintentionally changing the raw data. Also, automatically detects the schema for every file.

      Delimiter
      Edit and sync CSV files with GitHub directly in the browser.
